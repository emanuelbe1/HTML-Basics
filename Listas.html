Sobre las listas.

Hay dos tipos de listas que pueden ser definidas en HTML.

Las listas son:

  • Una lista desordenada, que comienza con el tag <ul>.
  • Una lista ordenada, especificada por el tag <ol>.

La lista anterior fue una lista desordenada. Cada elemento de la lista se especifica con el tag <li>.
