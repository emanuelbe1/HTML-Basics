Sobre los bloques.

-Definición básica-

Cúando cargamos un sitio web, todos los elementos HTML se organizan de manera predeterminada.
Casi todos los elementos HTML siguen alguno de estos patrones: pertenecen a los elementos de bloque o a los elementos en línea.
Esto es util para agrupar diferentes elementos y aplicar las mismas propiedades.

Elementos de Bloque

Los elementos de bloque HTML son aquellos que ocupan todo el ancho de la página, o del elemento que lo contiene.
Algunos ejemplos son:

  • <p>
  • <ul>
  • <h1> a <h6>

Elementos en Linea

Un elemento en línea HTML solo ocupa el ancho de su contenido.
Algunos ejemplos con los que he practicado hasta ahora:

  • <a>
  • <b>
  • <i>
  • <img>
